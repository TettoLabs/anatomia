# anatomia-workspace

TypeScript · Vitest

## Commands
- Build: `pnpm run build`
- Test: `pnpm run test -- --run`
- Lint: `pnpm run lint`
- Dev: `pnpm run dev`

## Conventions
- Functions: camelCase
- Files: lowercase
- Imports: relative
- Indentation: spaces, 2 wide

## Constraints
- camelCase for function names
- Run `pnpm run build` before committing
