# Anatomia

Verified AI development. Ship with proof.

Anatomia scans your project, detects your stack, conventions, and patterns, then generates AI-ready context files (`CLAUDE.md`, `AGENTS.md`, skills, gotchas) that make every AI coding tool smarter about your codebase.
