## Critical Rule

**The 25 static files have ZERO variables, ZERO placeholders, ZERO injection.**

If you see code doing string replacement on a mode file, that's wrong:

```typescript
// ❌ WRONG — Modifying static file
const modeContent = await fs.readFile(staticPath, 'utf-8');
const modified = modeContent.replace('{{framework}}', framework);
await fs.writeFile(destPath, modified);

// ✅ CORRECT — Verbatim copy
await copyAndVerifyFile(staticPath, destPath, fileName);
```

Only ENTRY.md template (dynamic file generated by setup complete) has variables.
